【遊戲目的】

真實社會縮影，30 種職業劇本 + 隨機人生事件。選擇你的身分，體驗不同人生的挑戰與機遇。

【遊戲規則】

  1. 選擇職業和體驗年份（1年/2年/3年）
  2. 每個月遇到一個抉擇，30秒內做選擇
  3. 追蹤三項數值：現金、健康、信用
  4. 任何一項降到 0 以下就失敗
  5. 撐過所有月份就成功

【計分方式】

• 綜合評分 = (現金×1.5) + (健康×1.0) + (信用×1.2)
• 生存加分 = 存活月數×5
• 職業難度係數：E(1.0x) / S(1.3x) / B(1.6x) / I(2.0x)
• 最終分數 = (綜合評分 + 生存加分) × 職業難度係數
